## Run
```console
$ sudo apt-get install curl
$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash
$ mkdir /home/ubuntu/apps
$ cd /home/ubuntu/apps
$ git clone https://github.com/nanulypartners/ninewatt.git
$ cd ninewatt
$ node app.js
```
